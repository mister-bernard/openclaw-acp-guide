# Contributing

This is a community guide. PRs are welcome.

## What we're looking for

- **Better orchestrator models** — Found something cheaper or smarter than Qwen 3.5 for routing? Let us know.
- **New gotchas** — Hit a wall we didn't document? Open an issue or PR.
- **Provider configs** — Got a working config for a provider we haven't covered? Add it to `examples/`.
- **Corrections** — If something in this guide is wrong or outdated, please fix it.

## How to contribute

1. Fork the repo
2. Make your changes
3. Open a PR with a brief description of what you changed and why

Keep it practical. This guide is for people who want to get ACP working, not for people who want to read a textbook.

## Code of conduct

Be helpful. Be kind. That's it.
